# 진동 통계 (Vibration Stats)

## 1. 분포 통계

가속도 데이터의 분포를 요약하는 값들.

### RMS (Root Mean Square)

```
RMS = √(평균(x²))
```

진동의 전체 에너지 크기를 나타내는 대표값. 개별 피크가 아닌 **전체적인 진동 강도**를 의미한다.
- 값이 클수록 전체적인 진동이 강함
- peak과 달리 순간 충격에 덜 민감

### Peak

```
Peak = max(|x|)
```

절대 최대값. 순간적으로 가장 큰 진동. 이상치(충격)에 민감하다.

### Min / Max

원시 값의 최소/최대. Peak과 달리 **방향(부호)을 포함**한다.

```
데이터: [-0.28, 0.01, 0.05, -0.02, 0.12]
Min = -0.28, Max = 0.12, Peak = 0.28
```

### Q1 / Median / Q3 (사분위수)

가속도 절대값을 작은 순서대로 정렬했을 때의 경계값.

```
정렬된 |가속도| 데이터 (12개):
0.01, 0.01, 0.02, 0.02, 0.03, 0.03, 0.04, 0.05, 0.07, 0.08, 0.12, 0.28
      ──────────┬──────────  ─────────┬─────────  ──────────┬──────────
           Q1 (25%)            Median (50%)           Q3 (75%)
           = 0.02              = 0.035                = 0.075
```

| 항목 | 위치 | 의미 | 활용 |
|------|------|------|------|
| Q1 | 하위 25% | 데이터의 75%가 이 값 이상 | 기저 진동 수준 |
| Median | 중앙 50% | 정중앙값 (평균보다 이상치에 강건) | 대표 진동 수준 |
| Q3 | 상위 25% | 데이터의 75%가 이 값 이하 | 높은 진동 수준 |

#### IQR (사분위 범위)

```
IQR = Q3 - Q1
```

데이터의 중앙 50%가 퍼진 정도. 진동의 **안정성** 지표.
- IQR 좁음 → 안정적 진동
- IQR 넓음 → 불안정 진동

#### 차트에서의 활용 (Vib 3Panel - Panel 1)

```
Q3 ───────────── 상단 밴드 경계
    ████████████ ← Q1~Q3 영역 (데이터의 50%가 이 안에)
Q1 ───────────── 하단 밴드 경계
Median ───────── 중앙 라인

밴드가 좁음 = 안정    밴드가 넓음 = 불안정
███                   █████████████
```

Q3가 급격히 높아지는 시점 → 설비 이상 징후

## 2. Threshold 초과 분석

기준값(기본 0.1g)을 초과하는 샘플에 대한 통계.

| 항목 | 계산 | 설명 |
|------|------|------|
| exceed_count | \|x\| > threshold인 샘플 수 | 고진동 발생 횟수 |
| exceed_ratio | exceed_count / 전체 샘플 수 | 고진동 비율 (0~1) |
| exceed_duration_ms | exceed_count / sample_rate × 1000 | 고진동 총 시간 |

```
가속도:  0.02  0.08  0.15  0.12  0.03  0.11  0.05
threshold ─────────────── 0.1g ───────────────────
초과 여부:  X     X     O     O     X     O     X
                exceed_count = 3
                exceed_ratio = 3/7 = 0.43
                exceed_duration_ms = 3ms (1000Hz 기준)
```

## 3. 이벤트 분류 (Burst vs Peak Impact)

threshold 초과 **연속 구간**의 지속 시간으로 분류.

```
가속도 파형:
              ┌────────────┐              ┌┐
──────────────┘            └──────────────┘└────────
              burst (≥500ms)             peak (<500ms)
              = 지속 진동                 = 순간 충격
```

| 이벤트 | 기준 | 의미 | 원인 예시 |
|--------|------|------|-----------|
| **Burst** | 연속 초과 ≥ 500ms | 지속적 진동 | 베어링 마모, 축 정렬 불량 |
| **Peak Impact** | 연속 초과 < 500ms | 순간 충격 | 이물질 충돌, 기어 결함 |

### 판별 로직

```python
for 각 연속_초과_구간:
    if 구간_길이 >= 500ms (= 500 샘플 @1000Hz):
        burst_count += 1
    else:
        peak_impact_count += 1
```

## 5. 평균(Mean) vs 중앙값(Median) vs RMS 비교

```
데이터: [0.01, 0.01, 0.02, 0.02, 0.03, 0.50]

Mean   = 0.098  ← 0.50(이상치)에 끌려감
Median = 0.02   ← 이상치 영향 없음
RMS    = 0.205  ← 에너지 기반, 큰 값에 가중치
```

- **Mean**: 이상치에 민감 → 진동 분석에 부적합한 경우 있음
- **Median**: 이상치에 강건 → 대표값으로 적합
- **RMS**: 에너지 관점 → 설비 진동 평가 국제 표준(ISO 10816)에서 사용

## 6. 분포 vs 사분위수

**분포**는 데이터 전체의 모양이고, **사분위수**는 그 분포를 숫자 3개로 요약한 것.

```
분포 (히스토그램):

빈도
 8 │  ████
 6 │  ████ ████
 4 │  ████ ████ ████
 2 │  ████ ████ ████ ████           ██
   └──────────────────────────────────
   0.01  0.03  0.05  0.07  0.09  0.28

→ "왼쪽에 몰려있고 오른쪽 꼬리가 길다" = 분포의 모양
```

사분위수는 이 분포를 숫자 3개로 요약:
```
Q1=0.025    Median=0.045    Q3=0.075
```

| 요약 방법 | 알 수 있는 것 | 알 수 없는 것 |
|-----------|--------------|--------------|
| 평균 + 표준편차 | 중심과 퍼짐 | 비대칭 여부 |
| Q1 / Median / Q3 | 중심 + 퍼짐 + 비대칭 | 꼬리 세부 모양 |
| 히스토그램 (분포) | 전부 | 숫자로 비교 어려움 |

사분위수를 쓰는 이유: DB에 히스토그램을 저장할 수 없지만 Q1/Median/Q3 3개 숫자는 저장 가능하고 비교도 쉽다.

## 7. 편차, 분산, 표준편차

### 편차 (Deviation)

각 데이터가 평균에서 얼마나 떨어져 있는가.

```
데이터: 2, 4, 4, 4, 5, 5, 7, 9
평균 = 5

편차 = 각 값 - 평균:
  2-5=-3, 4-5=-1, 4-5=-1, 4-5=-1, 5-5=0, 5-5=0, 7-5=2, 9-5=4
```

편차를 그냥 합하면 양수/음수가 상쇄되어 0이 된다. 그래서 제곱한다.

### 표준편차 구하는 과정

```
1. 평균                 Mean = 5
2. 편차                 -3, -1, -1, -1, 0, 0, 2, 4
3. 편차 제곱             9, 1, 1, 1, 0, 0, 4, 16
4. 제곱의 평균 = 분산    Variance = 32/8 = 4
5. 제곱근 = 표준편차     σ = √4 = 2
```

정리:
```
편차     = 값 - 평균               개별 거리
분산     = 편차²의 평균             퍼짐 (단위가 제곱)
표준편차  = √분산                  퍼짐 (원래 단위)
```

### RMS와 표준편차의 관계

```
표준편차 = √(평균((x - mean)²))   ← 평균 기준 퍼짐
RMS     = √(평균(x²))            ← 0 기준 크기
```

진동 데이터는 평균이 0에 가까우므로(양음 진동) **RMS ≈ 표준편차**가 된다.
이것이 진동 분석에서 RMS를 표준편차 대신 쓰는 이유.

## 8. 정규분포에서 사분위수 → 표준편차 역산

정규분포를 가정하면 Q1/Q3 위치가 고정되어 있다:

```
정규분포:
                    ┌───┐
                 ┌──┤   ├──┐
              ┌──┤  │   │  ├──┐
          ┌───┤  │  │   │  │  ├───┐
     ─────┤   │  │  │   │  │  │   ├─────
       -0.6745σ       0       +0.6745σ
          Q1        Mean        Q3
```

- Q1 = Mean - 0.6745 × σ
- Q3 = Mean + 0.6745 × σ

따라서:
```
σ = IQR / 1.349 = (Q3 - Q1) / 1.349
```

### 0.6745는 어디서 나온 값인가

정규분포의 **누적분포함수(CDF)의 역함수** 값.
"전체의 25% 지점이 평균에서 몇 σ 떨어져 있는가?"의 답.

```python
from scipy.stats import norm
norm.ppf(0.25)  # → -0.6744897...
norm.ppf(0.75)  # → +0.6744897...
```

주요 z값:

| 누적확률 | z값 | 의미 |
|----------|-----|------|
| 2.5% | -1.96σ | 95% 신뢰구간 하한 |
| 25% | -0.6745σ | Q1 |
| 50% | 0 | Median = Mean |
| 75% | +0.6745σ | Q3 |
| 97.5% | +1.96σ | 95% 신뢰구간 상한 |

### 주의

진동 데이터는 정규분포가 아닌 경우가 많다 (평소 잔잔 + 간헐적 큰 충격 = 오른쪽 꼬리 긴 분포).
이 공식으로 구한 σ는 실제보다 과소평가될 수 있다.


분포 (히스토그램):

빈도
 8 │  ████
 6 │  ████ ████
 4 │  ████ ████ ████
 2 │  ████ ████ ████ ████           ██
   └──────────────────────────────────
   0.01  0.03  0.05  0.07  0.09  0.28
                가속도 (g)

→ "왼쪽에 몰려있고 오른쪽 꼬리가 길다" = 분포의 모양
사분위수는 이 분포를 숫자 3개로 요약:


Q1=0.025    Median=0.045    Q3=0.075
   │             │              │
   ▼             ▼              ▼
   ├─── 25% ────┼──── 25% ────┼──── 25% ────┼── 25% ──┤
  0.01         0.025         0.045         0.075       0.28
비유하면:

비유
분포	반 전체의 성적표 (모든 점수)
사분위수	"하위 25%는 60점, 중간은 75점, 상위 25%는 88점" (요약)
평균	"반 평균 73점" (숫자 1개)
표준편차	"점수가 평균에서 ±12점 퍼져있다" (퍼짐 정도)
요약 방법마다 알 수 있는 것이 다릅니다:

요약 방법	알 수 있는 것	알 수 없는 것
평균 + 표준편차	중심과 퍼짐	비대칭 여부
Q1 / Median / Q3	중심 + 퍼짐 + 비대칭	꼬리 세부 모양
히스토그램 (분포)	전부	숫자로 비교 어려움
사분위수를 쓰는 이유는 분포 전체를 저장하지 않고도 데이터의 특성을 파악할 수 있기 때문입니다. DB에 히스토그램을 저장할 수는 없지만, Q1/Median/Q3 3개 숫자는 저장 가능하고 비교도 쉽습니다.
